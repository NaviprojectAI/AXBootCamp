# 이 강의를 시작하게 된 이유

**같은 업무, 다른 과정 — Claude.ai vs Claude Code**

| 상황 | Claude.ai | Claude Code |
|---|---|---|
| 엑셀 30개 파일을 합쳐서 요약 | 직접 선택해서 업로드 → 결과 다운로드 → **파일 정리는 내 몫** | **"이 폴더 전부 합쳐서 요약해"** → 결과도 폴더에 바로 저장 |
| 프로젝트 산출물 관리 | 생성할 때마다 **다운로드 → 정리 → 또 생성 → 또 다운로드** 반복 | **"산출물 관리해"** → 자동 생성·추적·관리 |
| 매주 같은 양식으로 보고서 작성 | 매번 양식과 맥락을 **다시 설명** | **"보고서 써"** → 데이터만 바꿔서 실행 |
| 대시보드 같은 앱 실행 | 코드 생성 → **Artifact 미리보기만 가능** (로컬 실행은 직접 세팅) | **"실행해"** → 브라우저에서 바로 확인 |
| 만든 서비스를 외부에 배포 | **불가능** — 서버 세팅·빌드·배포 전부 직접 | **"배포해"** → 빌드부터 배포까지 한 번에 |

> Claude Code를 활용하면 **업무 자동화의 범위가 확장**됩니다

# 클로드 코드 전환 기대 효과

| 효과 | 핵심 |
|---|---|
| **내 폴더에서 연속 작업** | 파일 업로드·맥락 재설명 없이, 프로젝트 폴더 안에서 바로 이어서 작업 |
| **반복 작업 자동화** | 매번 다시 설명할 필요 없이, 한 번 설정하면 데이터만 바꿔서 실행 |
| **만들고 바로 실행** | Artifact 미리보기가 아닌, 실제 환경에서 실행·빌드·배포까지 |
| **업무별 독립 공간** | 하나의 계정이라도 프로젝트마다 맥락이 분리 — 지식이 섞이지 않음 |
| **5배 큰 컨텍스트** | 대화가 끊기지 않아 복잡한 작업도 한 세션에서 완료 |


> 채팅형 AI가 "만들어주는 도구"라면, Claude Code는 "함께 일하는 환경"입니다

# 클로드 코드 도입 사례 — 나라장터 입찰공모 자동화

**AS-IS — 기존 업무 방식**
- 나라장터에서 키워드별로 직접 검색, 결과를 수동으로 정리
- 관심 공고마다 상세 페이지를 직접 클릭하여 자료를 다운하고 별도로 수집
- 상세 페이지 링크 취득 불가 → 나중에 다시 보려면 처음부터 검색해야 함

**TO-BE — Claude Code로 자동화**
- 클로드 코드한테 자동화 프로그램을 만들어 달라고 의뢰 → 완성된 프로그램으로 업무 처리
- 클로드 코드에서 "오늘 입찰공고 검색해서 시트에 정리해줘" 한 마디로 끝
- 상세 링크 취득 후 저장 — 공고 상세 페이지에 바로 접근 가능

# 팀에서 같은 계정을 쓴다면?

**Claude.ai / Desktop — 계정에 모든 게 묶임**

- 설정(MCP, 프롬프트 등)이 클라우드에 저장 → 한 명이 바꾸면 전원에게 영향
- 대화 기록이 섞임 — 누가 뭘 했는지 구분 어려움
- 프로젝트별 분리 불가

**Claude Code — 각자 PC에서 독립 작업**

- 설정(CLAUDE.md, MCP, Skill)이 로컬에 저장 → 서로 영향 없음
- 대화·작업 이력이 각자 터미널에 독립
- 프로젝트 폴더별로 완전히 분리

> 같은 계정, 같은 요금제 — 하지만 Claude Code는 **각자의 작업 환경**이 보장됩니다

# Claude 제품군 한눈에 비교

| | Claude.ai (웹) | Claude Desktop (Cowork) | Claude Code |
|---|---|---|---|
| **대상** | 누구나 | 일반 사용자 | 개발자 → **비개발자도 사용하는 추세** |
| **형태** | 웹 브라우저 | 데스크톱 앱 | CLI / VSCode 확장 |
| **파일 수정** | ❌ | ✅ 폴더 권한 설정 시 | ✅ 자유롭게 |
| **코드 실행** | ❌ | ⚠️ 제한적 (가상 환경) | ✅ 터미널 직접 실행 |
| **MCP 연동** | ❌ | ✅ Extensions로 간편 설치 | ✅ "연결해줘" 한마디 |
| **자동화 수준** | 1회성 대화 | 예약 작업, 멀티스텝 | Skill + Hook + MCP 조합 |

> 같은 클로드 두뇌, 하지만 **몸(도구)**이 다릅니다
> Cowork는 "쉽고 안전하게", Claude Code는 "강력하고 자유롭게"
