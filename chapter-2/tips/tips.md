## 팁 목록

### 1. 새 프로젝트 시작 시 가장 먼저 할 일 — CLAUDE.md 만들기
- 클로드 코드는 대화 시작 시 작업 디렉토리의 `CLAUDE.md`를 자동으로 읽는다
- 새 팀원에게 온보딩 문서를 주는 것과 같은 역할
- CLAUDE.md가 없으면 클로드 코드는 매번 맥락 없이 시작하게 된다
- **상황별 만드는 방법:**
  - 기존 프로젝트에 클로드 코드 도입 → `/init` 실행 (파일 구조와 설정을 자동 분석해서 생성)
  - 완전히 새로운 빈 프로젝트 → CLAUDE.md를 직접 작성하거나, 코드를 어느 정도 생성한 뒤 `/init` 실행
- **CLAUDE.md에 담아야 할 내용:**
  - 프로젝트 개요 (목적, 기술 스택)
  - 폴더 구조 및 주요 디렉토리 역할
  - 코딩 컨벤션 (네이밍 규칙, 스타일 가이드)
  - 자주 쓰는 명령어 (빌드, 테스트, 실행)
  - 작업 규칙 ("커밋 전 확인 사항", "건드리지 말 것" 등)
- 처음부터 완벽할 필요 없다. 프로젝트를 진행하면서 계속 다듬어가면 된다
