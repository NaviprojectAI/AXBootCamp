# 입찰공고 키워드 검색

나라장터 입찰공고를 키워드로 검색합니다.

## 실행 방법

1. 사용자가 지정한 키워드로 백엔드 API를 호출합니다:
   - `POST http://localhost:8000/api/search/run-single?keyword={키워드}`

2. 검색 결과를 아래 표 형식으로 정리합니다:

| 공고번호 | 공고명 | 발주기관 | 제출기한 | 금액(원) | 구분 |
|----------|--------|---------|---------|---------|------|

3. 결과 요약을 제공합니다:
   - 총 검색 건수 (새 공고 / 중복)
   - 마감 임박 공고 (3일 이내) 하이라이트

## 키워드 미지정 시
사용자에게 어떤 키워드로 검색할지 물어봅니다.
등록된 키워드 목록은 `GET http://localhost:8000/api/keywords`에서 확인할 수 있습니다.
