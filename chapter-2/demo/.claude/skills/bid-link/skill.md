# 입찰공고 상세 페이지 링크 조회

특정 입찰공고의 나라장터 상세 페이지 링크를 조회합니다.

## 실행 방법

1. 사용자가 말한 공고명 또는 키워드로 검색 결과를 조회합니다:
   - `GET http://localhost:8000/api/search/results?keyword={키워드}`

2. 검색 결과에서 `bid_url` 필드를 추출합니다.
   - 나라장터 직링크 형식: `https://www.g2b.go.kr/link/PNPE027_01/single/...`

3. 결과를 아래 형식으로 제공합니다:

   **[공고명]**
   - 발주기관: OOO
   - 제출기한: YYYY-MM-DD
   - 금액: OOO원
   - 상세 페이지: [나라장터 바로가기](URL)

4. 여러 건이 매칭되면 번호 목록으로 보여주고 사용자가 선택하게 합니다.

## 공고명 미지정 시
사용자에게 어떤 공고의 링크를 찾을지 물어봅니다.
