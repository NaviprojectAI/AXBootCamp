# 오늘의 입찰공고 검색 결과 보고서

오늘 검색된 입찰공고를 요약 보고서로 정리합니다.

## 실행 방법

1. 오늘 검색 결과를 조회합니다:
   - `GET http://localhost:8000/api/search/results/today`

2. 아래 구조의 마크다운 보고서를 생성합니다:

   ```
   # 입찰공고 일일 보고서 (YYYY-MM-DD)

   ## 요약
   - 총 검색 건수: N건
   - 신규 공고: N건
   - 마감 임박 (3일 이내): N건

   ## 키워드별 현황
   | 키워드 | 건수 |
   |--------|------|

   ## 주요 공고 목록
   (금액 상위 5건)

   ## 마감 임박 공고
   (제출기한 3일 이내)
   ```

3. 보고서를 `demo/reports/daily_YYYY-MM-DD.md` 파일로 저장합니다.
